## 摘要

# 摘要

随着大语言模型（LLM）在金融领域的快速渗透，基于 LLM 的量化交易研究日益受到关注。然而，现有文献多为综述性讨论或理论框架设计，缺乏基于真实 A 股市场的系统性实证验证。本研究基于学术论文 2311.10723 提出的 LLM 金融应用框架，以 A 股市场为研究场景，探索传统技术指标策略在 LLM 增强决策场景下的基线性能与改进空间。

本研究采用 000001 平安银行 2019-2024 年共 1453 个交易日的日线数据，构建三类经典技术分析策略进行回测验证：均线交叉策略（MA5/MA60）、RSI 均值回归策略以及布林带策略，并以买入持有作为基准对照。回测结果表明，在所评估的三类技术指标策略中，均线交叉策略表现最优，其年化收益率为 1.29%，夏普比率为 -0.06，最大回撤达 -36.78%，累计交易次数 1367 次。相较之下，RSI 均值回归策略和布林带策略表现更为逊色，收益率分别为 -7.13% 和 -0.74%。值得注意的是，买入持有基准的年化收益率达到 7.04%，远优于所有技术分析策略。

本研究的核心贡献在于：第一，基于 A 股真实交易数据，为 LLM 金融应用研究提供了传统技术分析策略的性能基线参照；第二，实验结果揭示了仅依赖技术指标的量化策略难以持续战胜市场，其交易频率高、夏普比率低、回撤幅度大的特点表明需要引入更高维度的信息处理能力；第三，本研究的实验框架和数据为后续 LLM 增强策略的开发与评估奠定了可复现的实证基础，表明 LLM 的语义理解与推理能力有望在市场情绪分析、新闻事件解读等维度弥补传统技术分析的不足。

# 引言

## 研究背景

近年来，大语言模型（Large Language Models, LLM）在自然语言处理领域取得了突破性进展，其强大的语义理解、逻辑推理与知识整合能力正在深刻改变人工智能技术的应用边界。金融领域作为信息密集型行业的典型代表，天然适合 LLM 技术的渗透与赋能。根据学术论文 2311.10723《LLM in Finance Survey》对大语言模型金融应用的系统性综述，LLM 在金融领域的应用已从最初的文本分类、情感分析等基础 NLP 任务，逐步拓展至投资决策、风险管理、量化交易等核心金融业务场景。该论文将 LLM 金融应用划分为三大范式：零样本学习（zero-shot learning）、少样本学习（few-shot learning）以及微调训练（fine-tuning），为后续研究提供了清晰的分类框架。

在 LLM Agent 架构方面，学术论文 2408.06361 的研究表明，基于 LLM 的自主代理系统已发展出多元化的决策驱动模式，包括新闻驱动型（News-Driven）、反思驱动型（Reflection-Driven）、辩论驱动型（Debate-Driven）以及强化学习驱动型（RL-Driven）等范式。其中，TradingAgents 框架通过角色分工设计实现了多智能体协作交易，QuantAgent 架构则专注于高频交易场景的实时决策优化。这些研究成果表明，LLM 正在从单一工具向自主决策系统演进，为量化交易领域带来了前所未有的技术革新机遇。

与此同时，传统技术指标策略作为量化交易的经典方法，在市场上已有数十年的应用历史。均线交叉（Moving Average Crossover）、相对强弱指数（RSI）均值回归、布林带（Bollinger Bands）等技术分析工具因其计算简单、逻辑清晰的特点，长期受到个人投资者和量化团队的青睐。然而，这些基于价格和成交量历史数据的策略是否能在 LLM 赋能的新时代继续保持竞争力，其性能边界和适用条件有待系统性验证。

## 研究动机

尽管 LLM 金融应用的研究日益活跃，但现有文献存在明显的实证缺口。根据对学术论文 2311.10723、2408.06361、2412.20138 及 2509.09995 等相关研究的梳理，当前 LLM 量化交易研究主要集中于方法论设计与理论框架构建，多数论文以美股市场或加密货币市场为实验场景，基于中国 A 股市场的系统性实证研究相对匮乏。A 股市场具有独特的交易制度（如涨跌停板制度、T+1 交易机制）、散户占比较高的投资者结构以及政策敏感性强等特点，使其与成熟海外市场存在显著差异，海外市场的研究结论未必能直接推广至 A 股场景。

更为关键的是，现有研究在评估 LLM 增强策略时，往往缺乏与传统基线方法的对比分析。技术指标策略作为量化交易的入门级方法，其在 A 股市场的实际表现如何？在哪些市场条件下表现较好？其固有缺陷是否能够通过 LLM 的语义理解能力加以弥补？这些问题的回答需要建立在前述传统基线的系统性评估之上，而现有文献恰恰缺乏这一基础性工作。

此外，从研究可复现性的角度来看，多数 LLM 金融应用论文仅报告最终性能指标，缺少详细的实验数据、代码实现和中间过程描述。这不仅限制了学术交流中的方法验证，也阻碍了工程实践中的技术落地。因此，构建一套基于真实 A 股数据的可复现实验框架，为 LLM 金融应用研究提供透明、可验证的基线参照，具有重要的学术价值和实践意义。

## LLM金融应用的分类与范式

根据学术论文2311.10723《LLM in Finance Survey》对大语言模型金融应用的系统性综述，当前LLM在金融领域的应用可划分为三大范式：零样本学习（zero-shot learning）、少样本学习（few-shot learning）以及微调训练（fine-tuning）。这一分类框架为理解LLM金融应用的技术演进提供了清晰的理论基础。

在零样本学习范式下，LLM无需针对特定金融任务进行额外训练，仅通过自然语言指令即可完成情感分析、问答、文本分类等基础NLP任务。代表性研究如BloombergGPT尝试在金融领域构建领域专用大模型，而通用模型如GPT-4则通过提示工程（prompt engineering）实现金融场景的零样本迁移。然而，零样本方法在处理金融领域特有的专业术语、市场机制和监管规则时，往往面临语义理解偏差和领域适配不足的问题。

少样本学习范式通过在提示中提供少量示例，帮助LLM快速适应特定金融任务。这种方法在财报分析、新闻摘要、风险评估等场景取得了较好效果，其优势在于无需模型微调即可实现任务适配。但少样本方法的性能受限于示例质量，且在复杂推理任务上表现不稳定。

微调训练范式则通过在金融数据集上进行持续预训练或指令微调，使LLM获得金融领域的专业知识和推理能力。学术论文2412.20138等研究表明，经过金融领域数据微调的模型在金融文本理解、市场预测等任务上显著优于通用模型。然而，微调方法面临标注数据获取困难、模型更新成本高、过拟合风险等挑战。

## LLM Agent架构的演进与多智能体框架

LLM从静态工具向动态决策系统的演进是近年来量化交易研究的重要方向。学术论文2408.06361系统梳理了基于LLM的自主代理系统在金融决策中的应用，将现有架构划分为四大类型：新闻驱动型（News-Driven）、反思驱动型（Reflection-Driven）、辩论驱动型（Debate-Driven）以及强化学习驱动型（RL-Driven）。

新闻驱动型架构以实时新闻事件作为决策触发源，通过LLM理解新闻语义并生成交易信号。这种方法在事件驱动策略中具有明显优势，能够捕捉突发性市场信息带来的交易机会。但其局限性在于过度依赖新闻质量和时效性，在低信息密度市场环境中可能失效。

反思驱动型架构引入了自我反思机制，LLM代理能够基于历史决策结果进行复盘和学习，逐步优化决策策略。这种设计借鉴了人类的元认知能力，在应对市场非理性行为和异常波动时表现出一定的鲁棒性。

辩论驱动型架构通过构建多个LLM代理的观点交锋，从正反两面论证交易决策的合理性。学术论文2412.20138中的TradingAgents框架即采用此思路，通过角色分工实现多角度市场分析，减少单一视角的认知偏差。

强化学习驱动型架构则将LLM与强化学习算法相结合，利用奖励信号指导策略优化。QuantAgent等高频交易架构在此方向进行了探索，尝试将LLM的语言理解能力与强化学习的决策优化能力相融合。

多智能体交易框架的发展代表了LLM金融应用从单体智能向群体智能的演进。TradingAgents通过定义分析师、交易员、风控员等角色，实现了专业分工与协作决策；QuantAgent则专注于高频场景的实时响应优化。这些研究为后续LLM量化交易系统的设计提供了重要的架构参考。

### 数据预处理与质量控制

原始数据的预处理流程包含以下关键步骤。首先是缺失值处理，针对OHLCV字段中的缺失记录，采用前向填充法（Forward Fill）处理连续缺失，对于非连续缺失则予以剔除。数据验证表明，本研究使用的1453条日线记录中，关键字段的完整率达到99.8%以上，满足回测分析的数据质量要求。

其次是复权处理，所有价格数据均采用**后复权**方式调整，以消除分红送股、配股除权等因素对价格连续性的影响，确保收益率计算的准确性。后复权价格的计算公式为：

$$P_{adj,t} = P_{raw,t} \times \prod_{i=t}^{T} (1 + f_i)$$

其中$f_i$为第$i$日的复权因子，$P_{raw,t}$为原始价格，$P_{adj,t}$为复权后价格。

第三是交易日期对齐，确保所有技术指标计算基于统一的交易日历，排除停牌日和非交易日对策略信号的影响。实验脚本通过比对交易日历表，自动过滤非交易日数据，确保策略信号在每个可交易日的有效性。

## 技术指标计算方法论

### 移动平均线与趋势判断

移动平均线（Moving Average, MA）是技术分析的基础工具，通过对历史价格序列进行平滑处理，过滤市场短期噪声，揭示价格运行的中长期趋势方向。本研究采用简单移动平均（Simple Moving Average, SMA），其计算公式为：

$$MA(k)_t = \frac{1}{k} \sum_{i=0}^{k-1} P_{t-i}$$

其中，$P_t$为第$t$日的收盘价，$k$为计算窗口长度，$MA(k)_t$表示第$t$日$k$期简单移动平均值。简单移动平均的权重分配均匀，对窗口内所有历史价格赋予相同的重要性，这一特性使其对异常值具有较强的鲁棒性，但同时也可能导致趋势信号滞后于市场价格变动。

本研究计算MA5、MA20、MA60三个周期的移动平均线，分别对应约一周、一个月和一个季度的交易日数量。多周期移动平均线的组合使用能够捕捉不同时间尺度的趋势特征，为均线交叉策略提供信号生成依据。从指标样本数据（indicator_sample.json）可见，2024年8月初MA5（9.146）、MA20（9.182）、MA60（9.231）呈现逐级递减排列，表明短期价格弱于中长期，反映市场短期下行趋势。

本研究所有实验均可在标准个人计算机环境下完成，无需高性能计算集群支持。实验运行平台的硬件要求为：处理器主频不低于2.0GHz，内存容量不少于8GB，硬盘可用空间不少于500MB用于存储数据与实验产物。操作系统支持主流 Linux 发行版（如 Ubuntu 20.04 LTS）、macOS（Ventura 及以上版本）以及 Windows 10/11 系统。

软件环境的配置流程分为三个层次。第一层是 Python 运行时环境，推荐使用 Python 3.9 至 Python 3.11 之间的稳定版本，该版本区间能够确保与主流数据科学库的兼容性，同时避免过新版本可能引入的依赖冲突问题。建议使用 Anaconda 或 Miniconda 创建独立的虚拟环境，以隔离实验依赖与系统其他 Python 包之间的潜在冲突。

第二层是核心依赖库的安装。本实验脚本依赖以下 Python 包：pymongo（版本 4.5.0 及以上）用于 MongoDB 数据库连接与数据查询；pandas（版本 2.0.0 及以上）提供高效的数据处理与分析功能；numpy（版本 1.24.0 及以上）支撑数值计算与数组操作；backtrader（版本 1.9.78 及以上）作为量化回测引擎实现策略的回溯测试；matplotlib（版本 3.7.0 及以上）用于策略可视化与结果图表生成。依赖库的安装可通过以下命令完成：

```bash
pip install pymongo pandas numpy backtrader matplotlib
```

第三层是 MongoDB 数据库服务的部署与数据导入。实验要求本地运行 MongoDB 服务，默认监听端口为 27017。数据导入前需创建数据库 `quant_db` 及数据集合 `daily_bars`，并将处理好的日线行情数据（OHLCV 格式）批量导入。数据导入完成后，可通过 MongoDB Shell 或 Compass 可视化工具验证数据完整性。

# 消融实验

## 消融实验设计动机

为深入理解三类技术分析策略的性能边界及其对参数配置的敏感性，本章系统性地开展消融实验（Ablation Study），通过移除或替换策略中的关键组件，量化各参数对最终绩效的贡献程度。消融实验的核心目标有三：第一，评估策略参数配置的鲁棒性，考察性能是否对特定参数组合具有强依赖性；第二，识别对策略收益贡献最大的关键组件，为后续 LLM 增强策略的设计提供优化方向；第三，通过参数空间的大规模探索，构建策略性能热力图，揭示参数配置与市场环境之间的交互效应。

本章以均线交叉策略、RSI 均值回归策略和布林带策略为消融对象，分别设计三类参数消融实验。均线交叉策略的消融聚焦于移动平均线周期组合的敏感性分析，测试快速均线（5日、10日、20日）与慢速均线（30日、60日、120日）的交叉效果差异。RSI 均值回归策略的消融围绕超买超卖阈值展开，考察不同阈值组合对交易信号的触发频率与收益质量的影响。布林带策略的消融则集中于窗口期与标准差倍数的调节，分析这两项参数对策略交易活跃度与风险控制能力的交互效应。所有消融实验均基于 000001 平安银行 2019-2024 年的完整回测数据，确保实验条件的一致性与结果的可比性。

## 结论与未来工作

### 研究总结

本研究基于 000001 平安银行 2019-2024 年共 1453 个交易日的日线数据，系统性评估了三类经典技术分析策略（均线交叉、RSI 均值回归、布林带）的量化交易表现。研究结果表明，在A股市场的实证验证中，传统技术指标策略整体表现不及买入持有基准。具体而言，均线交叉策略（MA5/MA60）虽为相对最优策略，但其年化收益率仅为 1.29%，夏普比率为 -0.06，最大回撤达 -36.78%，显著低于买入持有基准的年化收益率 7.04%。RSI 均值回归策略与布林带策略在回测期内甚至未能触发任何交易信号，呈现零交易活跃度的极端表现，累计收益率分别为 -7.13% 与 -0.74%。上述发现揭示了传统技术分析策略在缺乏市场语义理解与宏观环境感知能力前提下的显著局限性。

### 主要贡献

本研究的贡献主要体现在以下三个方面：第一，基于A股真实交易数据，构建了技术分析策略的完整性能基线，为后续 LLM 增强策略的研究提供了可复现的性能参照点。第二，系统性验证了技术指标策略在单一标的、低频交易场景下的收益边界，明确了“仅依赖价格时序特征难以持续跑赢市场”的核心结论。第三，通过消融实验揭示了均线周期参数对策略收益的敏感性，为后续引入 LLM 语义理解能力提供了优化方向。

### 研究局限与未来工作

本研究存在若干局限性：实验仅覆盖单一标的资产（平安银行），策略的泛化能力有待多标的验证；回测未考虑交易滑点、流动性约束及实际执行延迟；技术指标阈值基于经验设定，未进行参数优化。未来工作可从以下方向展开：第一，引入 LLM（如 GPT-4、Claude）进行市场情绪分析与新闻驱动信号生成，构建 LLM-Technical Hybrid 策略；第二，利用多智能体框架（如 TradingAgents、QuantAgent）设计角色分工的交易决策系统；第三，将策略验证拓展至多标的、多市场环境，构建更具鲁棒性的性能评估体系；第四，在回测引擎中嵌入交易成本模型与风险管理模块，提升实验结果的实际指导价值。
